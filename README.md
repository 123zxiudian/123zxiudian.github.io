## 🧠 About us

我们是隶属于贵州大学省部共建公共大数据国家重点实验室大数据分析与决策研究室，专注于**人工智能技术的应用与实践**。团队以研究生为主，在指导教师的带领下开展探索性研究与工程实现。

### 🔍 研究兴趣
- 大语言模型（LLM）的微调、推理优化与本地部署  
- 时间序列分析与预测  
- 智能问答系统与教育辅助工具开发  
- 轻量化模型在边缘设备上的应用

### ⚙️ 技术栈
- **语言**：Python（主力）、C++（部分模块）
- **框架**：PyTorch、Hugging Face Transformers、LangChain
- **工具链**：Git / Docker / Linux / ONNX / FastAPI
- **部署**：本地服务器 + 边缘设备（如 Jetson、树莓派）

### 🎯 目标与理念
- 坚持“**做中学**”：通过真实项目驱动能力成长  
- 注重**工程落地**：不仅跑通实验，更要可部署、可复现  
- 鼓励**开源协作**：代码规范、文档完整、版本清晰  
- 培养**独立思考与解决问题**的能力

### 🤝 对外合作
- 欢迎校内对 AI 应用感兴趣的同学参与项目实践  
- 支持本科生开展毕业设计或科研训练（需通过指导教师联系）  
- 所有开源项目均欢迎 Issue 与 PR（请遵守 CONTRIBUTING 规范）
